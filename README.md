# Process Mining Analysis – Travel Expense Reimbursement

This repository contains a team university project focused on process mining analysis of the travel expense approval and reimbursement process.

## Project overview

The project analyzes a business process related to travel request approval and travel expense reimbursement using event log data. The focus of the analysis was on identifying:

* bottlenecks
* delays
* deviations in process execution
* opportunities for process improvement

The project also included event log quality checks and comparison of process behavior based on available process data.

## Methods and tools

* process mining
* Disco
* event log analysis
* process performance analysis
* process analysis and interpretation
